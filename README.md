## Usage

To integrate with the Webasto Unite using Modbus TCP, ensure that you have the correct address and data type as outlined above. Adjust your application accordingly to read from and write to the specified registers. 

Make sure to consult the official Webasto documentation for any additional information regarding register mappings and operational details.
